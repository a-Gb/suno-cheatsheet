Technical Parameters

Parameter Description Values Default Status
temperature Controls the randomness/creativity of the output. Higher values produce more varied results. 0.0 to 1.0 0.8 Official
guidance Controls how closely the model follows your prompt. Higher values adhere more closely. 1.0 to 10.0 6.0 Official
seed Sets a specific random seed for deterministic generation. Same seed + same parameters = same result. Any integer Random Official
length Controls the length of the generated track in seconds. 30, 60, 120, 180, 240 120 Official
key_signature Musical key for the composition. C, C#, D, D#, E, F, F#, G, G#, A, A#, B (with major/minor) Auto Official
tempo Beats per minute - controls the speed of the track. 40 to 200 Auto Official

Additional Parameters

Parameter Description Values Default Status
time_signature Time signature of the track. 4/4, 3/4, 6/8, etc. 4/4 Experimental
style_reference A reference song or artist to influence the style and sound of the generation. String with artist name and song title None Experimental
diversity Controls the diversity of the output. Higher values lead to more diverse results. 0.0 to 1.0 0.5 Experimental
energy Controls the energy level of the track. Higher values create more intense, dynamic pieces. 0.0 to 1.0 0.7 Experimental

Structure Tags

Section Tags

Tag Description Example Status
[VERSE] Marks the verse section of a song, typically containing the main story or narrative. [VERSE] Walking through the neon streets, memories flood back... Official
[CHORUS] Marks the chorus section, which is the most memorable and repeated part of the song. [CHORUS] Never forget where we started, always remember who we are... Official
[INTRO] Marks the introduction section of the song, setting the mood and tone. [INTRO] (Ambient city sounds with soft piano) Official
[OUTRO] Marks the final section of the song, often bringing closure. [OUTRO] As the lights dim and the crowd fades away... Official
[BRIDGE] Marks a contrasting section that typically appears after the second chorus. [BRIDGE] Breaking through the walls I've built around my heart... Official
[PRE-CHORUS] Marks a section that builds tension before transitioning into the chorus. [PRE-CHORUS] The pressure builds with every breath I take... Official
[HOOK] Marks a catchy, repeatable section designed to 'hook' the listener. [HOOK] Live fast, dream big, never look back! Official
[INSTRUMENTAL] Marks a section without lyrics, showcasing the instrumental elements. [INSTRUMENTAL] (Dramatic synth solo with pulsing beat) Official

Example Prompts

Basic Example

A chill lo-fi hip hop track with smooth jazzy piano and laid-back beats. Lyrics about late night creative sessions and finding inspiration in the quiet hours.

[VERSE]
Midnight oil burning bright
Thoughts flow freely in the night
Silence speaks the loudest now
Creativity shows me how

[CHORUS]
Quiet hours, loud mind
Leaving all the stress behind
In this space between the days
Inspiration finds its ways

This basic example uses a simple descriptive prompt followed by structured lyrics with VERSE and CHORUS tags. It provides a straightforward but effective framework for generating a cohesive lo-fi track.

Best Practices

Start With Clear Sonic References

Include specific reference points for the sound you want, such as artists, songs, or production styles rather than just genre names.

Tip: Instead of 'make an electronic track', try 'Create an electronic track with warm analog synths like Tycho, tight drums like Bonobo, and atmospheric textures similar to Jon Hopkins.'

Use Proper Section Capitalization

Udio's section tags use ALL CAPS format ([VERSE], [CHORUS]) for optimal recognition. This helps the model properly structure your song.

Tip: Always capitalize section tags completely: [VERSE] not [Verse], [CHORUS] not [Chorus], etc.

Optimize Parameter Combinations

The relationship between temperature and guidance significantly affects your results. Higher temperature with lower guidance creates more unexpected variations.

Tip: For highly creative but less predictable results, try temperature=0.9 with guidance=4.0. For more faithful adherence to your prompt, use temperature=0.6 with guidance=7.0.

Create Contrast Between Sections

Design your lyrics with distinct emotional tones or narrative perspectives between verses, choruses, and bridges to create musical contrast.

Tip: Try using more detailed, specific imagery in verses and more universal, emotional concepts in choruses to match how music typically flows.

Do's and Don'ts

Do

  • Use specific instrumental and production references
  • Balance technical description with emotional language
  • Keep lyrical sections reasonably sized (4-8 lines)
  • Maintain consistent themes throughout your lyrics
  • Use ALL CAPS for section tags
  • Include tempo and key information when you have specific preferences
  • Define both musical and emotional qualities in your prompt

Don't

  • Don't mix too many conflicting genre references
  • Avoid extremely detailed vocal requirements that are unrealistic
  • Don't write overly long sections (more than 8-10 lines)
  • Avoid leaving section tags incomplete or improperly formatted
  • Don't use extremely high guidance values (>9.0) which can cause stiff results
  • Avoid focusing only on lyrics without any musical direction
  • Don't request specific artists' exact voices by name
